Inline, inline-block, block

inline

h3 tag and p tag are by default block elements, but we can manually make them appear as inline elements

inline-block

The difference between inline and inline-block is that we can add width and height to inline-block elements whereas for inline elements we can't

block

Block elements take the full width available, and start on a new line. We can add width and height to block elements

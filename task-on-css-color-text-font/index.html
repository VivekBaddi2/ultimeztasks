CSS color

background-color property is used to set the background color of an element

border-color property is used to set the border color of an element

color property is used to set the text color of an element

Different types of color values are -

rgb - rgb(0,0,0) represents black color

hsl - hsl(217, 0%, 100%) represents white color

hex - #2200ff represents blue color

rgba - rgba(21, 255, 0, 0.5) represents color with opacity 0.5

hsla - hsla(65, 100%, 50%, 0.3) represents color with opacity 0.3

CSS text

color property is used to set the color of text

background-color property is used to set the background color of an element

text-align property is used to align the text to left right or center

text-decoration property is used to add underline or overline or a strike-through over the text

text-transform property can be used to transform the text entirely to uppercase or lowercase or just capitalise the first letter of each word

Spacing can be added between words using - word-spacing: - property

letter-spacing: - property can be used to add spacing between letters

line-height: - property can be used to add spacing between lines

text-shadow property enables shadow effects to be applied to the text

CSS font

font-family property is used to set the font of an element

font-size property is used to set the size of the font

font-style property is used to set the style of the font
